What is MapIt?

A zero-hassle hosted mapping service…

Call our public API to get information about any point on the globe. We look after versioning and updates, so you don’t have to.

…And a Django app for ultimate flexibility

Want to collate detailed local geographic data, and host it near to your customers? Use MapIt as a component in your own site.

The easy way to work with geographic data

Geographic data is complex, and changes regularly. Let MapIt do the hard work of querying and organising it all.

A powerful API

Both hosted and self-hosted versions of MapIt have a REST API that is fast and easy to use. Access your data as KML, GeoJSON, or HTML web pages.
